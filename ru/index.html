Платформа для back-office приложений на Laravel Framework

Это пакет для создания административных приложений. Которая абстрагирует общие функции, необходимые для бизнес-приложений, CMS и систем электронной коммерции.

Легко начать
Разработка ведётся на PHP, не обязательно использовать HTML, CSS или JavaScript. Всё чтобы приложения были просты в использовании.
Быстрая загрузка страниц
Производительность одностраничного приложения без дополнительной сложности, переходы без перезагрузки страницы или каких-либо ресурсов.
Поиск данных
Находите необходимую информацию с любой страницы. Интегрированный Laravel Scout позаботится о результатах поиска.
Фильтрация и сортировка
Создайте фильтры, чтобы пользователи могли быстро найти различные сегменты данных. Полностью контролируя SQL код.
Уведомления пользователей
Одноразовые и персональные уведомления будут отличным способом оповещать пользователей о том, что происходит в вашем приложении.
Управление ролями
Разграничение прав доступа в сочетании с политиками дают удобные возможности в разработке и поддержке.
Представляйтесь другим
У вашего клиента проблема? Вы сможете легко повторить её, аутентифицировашись в системе как один из ваших пользователей.
Two-Factor аутентификация
Защитите аккаунты клиентов с помощью двухфакторной аутентификации, использующей протокол TOTP, например, на мобильном устройстве.
